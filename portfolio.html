Restraunt Website

The restaurant website is a modern and responsive web template designed to showcase the best of your dining experience. It features a welcoming home page to highlight the ambiance and unique offerings of the restaurant, a detailed menu page with categories, item descriptions, and prices, and an about us section to share the restaurant's story and mission. Additionally, the contact page provides essential information like location, operating hours, and a form for reservations or inquiries. Built using HTML,CSS,and JavaScript, the website ensures a seamless and engaging user experience across all devices.

To Do List

This project is a web-based application designed to help users manage their daily routines effectively. It serves as a digital alternative to a traditional diary, allowing users to add, edit, and delete tasks as needed. The application is built using HTML,CSS and JavaScript, ensuring a user-friendly interface and responsive design. By organizing tasks in a clear and accessible manner, this ToDo-App aims to enhance productivity and keep users on track with their daily commitments.

Brand Page

The brand page is a sleek and modern web application developed using ReactJS, showcasing Nike shoes with a focus on style and functionality. The page features high-quality images of Nike footwear, designed to captivate visitors and highlight the brand's iconic products. It provides details about the shoes, including key features and benefits, and integrates links to shopping platforms where users can purchase them effortlessly. The ReactJS framework ensures a dynamic and seamless user experience, with responsive design elements that cater to all device types. This project is a perfect blend of aesthetic appeal and practicality, aimed at delivering an engaging shopping experience for users.
